If some properties have been defined for the same selector (element) in different style sheets, the value from the last read style sheet will be used.

note: use "inset" instead of "top", "right", etc

Use property "Inherit" for links instead of hardcoding

Make use of version numbers

Really learn how :after and :before pusedo works

Use Simple Selectors Use simple selectors when possible. Complex selectors increase the parsing time.

Avoid the universal selector (*) when not strictly necessary. The universal selector (*) affects every element and can slow down page rendering.

Always use the :focus pseudo-class to ensure that interactive elements (like links, buttons, input fields) have a clear visual focus style. E.g: a:focus, button:focus, input:focus { outline: 2px solid orange; }
